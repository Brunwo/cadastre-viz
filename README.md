

this app can be accessed at https://brunwo.github.io/cadastre-viz/ but without AI features (turn it off in UI )

you can load a list of parcels from a csv file and visualize them all at once on a map.
also export gpx file of the parcels : load in locus maps / your trace app of choice to navigate and explore parcels boundaries 

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
